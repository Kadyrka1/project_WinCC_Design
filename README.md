# project_WinCC_Design

A simple WinCC HMI screen simulating a small process system with two tanks (T1, T2), interconnected piping, valves (V1–V3), and a pump (P1). The interface allows manual/automatic control of flow, valve positions, and tank levels, with real-time visualization of fluid levels and status indicators. It also includes basic controls for operation modes, simulation, and process trend monitoring, along with an alarm panel for system events.
